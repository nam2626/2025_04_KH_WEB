width & height 예제

단위 : px

정확한 크기 지정, 뷰포트 크기에 상관 없이 항상 동일함

Width: 150px
Height: 80px
Width: 150px
Height: 80px
Width: 150px
Height: 80px

단위 : 퍼센트 (%)

부모 요소의 크기에 비례하여 크기가 결정됩니다. 부모 컨테이너의 크기를 조절해보세요.

Width: 30%
Height: 100px
Width: 30%
Height: 100px
Width: 30%
Height: 100px

단위 : rem 단위

HTML 루트 요소(`html`)의 `font-size`에 따라 크기가 결정됩니다. 기본적으로 1rem은 16px입니다.

Width: 8rem
Height: 5rem
Width: 8rem
Height: 5rem
Width: 8rem
Height: 5rem

단위 : 뷰포트 (vw, vh)

브라우저 뷰포트(화면)의 너비(`vw`)와 높이(`vh`)에 비례하여 크기가 결정됩니다. 브라우저 창 크기를 조절해보세요.

Width: 20vw
Height: 10vh
Width: 20vw
Height: 10vh
Width: 20vw
Height: 10vh
